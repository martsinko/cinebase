# CineBase SPA 🎬

Навчальний SPA-додаток на ReactJS для теми **«Каталог фільмів і серіалів»**.

## Що реалізовано

- SPA на ReactJS без перезавантаження сторінок
- Навігація між сторінками: головна, каталог, режисери, жанри, студії, редагування, авторизація
- Навчальна авторизація зі збереженням користувача в `localStorage`
- Компонент `LocalCounter` з власним станом і подіями (трекер переглядів)
- Завантаження даних з API у форматі JSON через AJAX `fetch`
- Відображення фільмів із підстановкою даних зі зв'язаних таблиць (жанр, режисер, студія)
- Фільтрування і сортування фільмів (за жанром, назвою, рейтингом, роком)
- Пошук фільмів за назвою
- Форма додавання/редагування фільму через `PUT`/`POST`-запити
- Видалення записів через `DELETE`-запити
- Управління станом через `useReducer`
- Красивий кінематографічний дизайн у темних тонах

## Структура файлів

```
cinebase/
├── index.html      — точка входу, підключає React і app.js
├── app.js          — весь React SPA
├── style.css       — стилі (темна кінематографічна тема)
├── index.php       — REST API (JSON-сховище або MySQL)
├── data.json       — демо-дані (JSON-сховище)
├── schema.sql      — SQL-схема для MySQL
├── Dockerfile      — для деплою на Render
├── render.yaml     — конфіг Render
└── start_spa.bat   — швидкий запуск на Windows
```

## Ресурси API

| Метод | URL                      | Дія                  |
|-------|--------------------------|----------------------|
| GET   | `/film`                  | Список фільмів       |
| GET   | `/film/{id}`             | Один фільм           |
| POST  | `/film`                  | Додати фільм         |
| PUT   | `/film/{id}`             | Оновити фільм        |
| DELETE| `/film/{id}`             | Видалити фільм       |
| GET   | `/director`              | Список режисерів     |
| GET   | `/genre`                 | Список жанрів        |
| GET   | `/studio`                | Список студій        |

## Локальний запуск (без бази даних)

### Варіант 1 — PHP вбудований сервер (рекомендовано)
```bash
php -S localhost:8080 index.php
```
Відкрий: `http://localhost:8080`

### Варіант 2 — Windows bat-файл
Запусти `start_spa.bat`

### Варіант 3 — XAMPP
1. Скопіюй папку `cinebase` в `htdocs`
2. Відкрий: `http://localhost/cinebase/`

> **Без MySQL** — API автоматично використовує `data.json`.

## Підключення MySQL (XAMPP)

1. Запусти MySQL в XAMPP
2. Імпортуй `schema.sql` через phpMyAdmin або:
   ```bash
   mysql -u root < schema.sql
   ```
3. Перезапусти PHP сервер

## Деплой на Render

1. Завантаж папку на GitHub
2. У Render → `New Web Service` → підключи репозиторій
3. Обери **Docker** environment
4. Деплой — готово!

Якщо MySQL не підключений, API автоматично використовує JSON-сховище.
